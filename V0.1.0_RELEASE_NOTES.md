## What's Included in v0.1.0

### Core Features
- 16 interactive MIDI modes (KEYS, BEATS, ZEN, DROP, RNG, XY PAD, ARP, GRID, CHORD, LFO, TB3PO, GRIDS, RAGA, EUCLID, MORPH, SLINK)
- Bluetooth MIDI output
- Hardware MIDI (DIN-5 connector)
- USB MIDI (ESP32-S3 only)
- ESP-NOW wireless MIDI networking
- Display autoscaling for different screen sizes
- Remote display via WiFi (optional)
- Screenshot capture to SD card

### Supported Hardware
- ESP32-2432S028R (CYD) - 2.8" 320x240
- ESP32-4832S035C - 3.5" 480x320 capacitive
- ESP32-4832S035R - 3.5" 480x320 resistive
- ESP32-4832S040R - 4.0" 480x320 resistive
- ESP32-S3 headless dongles with USB MIDI

### Installation Methods
- Web installer (browser-based, no software required)
- Pre-built firmware binaries
- Build from source (PlatformIO or Arduino IDE)
