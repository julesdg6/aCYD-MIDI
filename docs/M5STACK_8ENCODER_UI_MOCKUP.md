## Color Scheme

```
Background:       Black (#0000)
Encoder boxes:    Dark gray surface (#2945)
Encoder borders:  Cyan primary (#06FF)
Labels:           White (#FFFF)
Values (accent):  Bright cyan (#07FF)
Encoder numbers:  Dim gray (#8410)
Progress filled:  Cyan primary (#06FF)
Progress empty:   Black (#0000)
Error message:    Red (#F800)
Buttons:          Orange secondary (#FD20), Cyan accent, etc.
```

## Notes

- Layout automatically adapts to display resolution
- All coordinates use SCALE_X() and SCALE_Y() macros
- Touch targets are properly sized for finger input
- Visual feedback updates in real-time as encoders turn
- Connection status shown when hardware not detected
